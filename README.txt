### How to run the program:
Ensure that the correct MySQL Connector driver is used. Launch the program (assuming the correct Java specs
are used)and log in to the application. When directed to the MainScreen, choose a button to select an action
to either add,modify, or delete data for either the customer, appointment, or report options. Load MySQL to
view the changes made to the database.


### Additional report ran in part A3f:
The additional report displays all appointments based on the user's name selected from the combo box.
It displays the user ID, contact ID, appointment ID, appointment type, start and end times.
